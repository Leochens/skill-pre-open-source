# Contributing

Issues and pull requests are welcome.

Please keep changes focused on practical open-source readiness:

- Preserve the `SKILL.md` frontmatter format.
- Keep safety rules conservative around secrets, private files, deletion, history rewriting, and publishing.
- Do not add advice that encourages exposing credentials or private assets.
- Update both `README.md` and `README.zh-CN.md` when public-facing behavior changes.

When changing the checklist, make sure it remains useful for small solo projects as well as larger repositories.
